# BookVerse Frontend

Frontend React terpisah untuk backend BookVerse.

## Cara menjalankan

```bash
cd C:\bookwise\bookverse-frontend
npm install
copy .env.example .env
npm run dev
```

Pastikan backend jalan di:

```txt
http://localhost:5000
```

Buka frontend:

```txt
http://localhost:5173
```
